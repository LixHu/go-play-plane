# 飞机大战游戏

一个使用Go语言和Ebiten游戏引擎开发的经典飞机射击游戏。

## 游戏特点

- 精美的动画启动界面
- 两种游戏模式：关卡模式和无尽模式
- 多种敌机类型
- 丰富的武器升级系统
- 流畅的游戏体验

## 操作说明

- 方向键：移动飞机
- 空格键：发射子弹
- ESC键：返回菜单

## 开发环境

- Go 1.24
- Ebiten v2

## 启动方式

```bash
go run .
```
